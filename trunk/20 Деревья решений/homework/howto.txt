В этом домашнем задании вам надо провести обучение классификаторов с помощью деревьев решений. Классификаторов должно быть два, так как датасета тоже два. Можно использовать знания про деревья решений не только из 20 лекции, но и более поздних.

Датасеты находятся в папках HCF и RF. В папке HCF находится уже известный вам датасет.
В RF находится новый датасет.
Формат нового датасета:  

В файле learn.txt находятся данные для обучения. В файле test.txt находятся данные для тестирования.

Формат файлов:
HCF:
В каждой строке находятся результаты одного наблюдения. В первом столбце - id наблюдения, во втором столбце - номер класса, к которому отнесли наблюдение. Столбцы с третьего по последний это факторы на которых надо обучаться.

!!!ВНИМАНИЕ - формат второго файла отличается от старого.!!!
RF:
В каждой строке находится результат одного наблюдения. В первом столбце - id машины, во втором столбце - id наблюдения для конкретной машины, в третьем - факт перегрева двигателя. В остальных столбцах находятся значения различных датчиков автомобиля. Собственно факт перегрева и надо предсказывать с помощью классификатора.
В папке RF файлы заархивированы. Код должен работать с разархивированными файлами.

В качестве метрики качества предсказания полученного мультиклассификатора нужно использовать каппу Кохена. Эту метрику стоит выводить и для learn-а и для test-а.

Код и решение следует присылать на адрес "saintnik@yandex-team.ru".

В теме письма должно быть написано "Домашняя работа 20 неделя Иванов И.И.", где вместо Иванов И.И. надо поставить свою фамилию и инициалы.
Код должен быть в прикреплён к письме в виде zip файла. Название файла должно быть вида Ivanov.zip. Вместо Ivanov в названии файла должна быть транслитерация фамилии студента.

Код необязательно должен состоять из одного файла. Главное и единственное требование к коду - будучи помещён в папку с исходными файлами датасета он должен выполнять задание. Никаких дополнительных ручных манипуляций с данными и кодом не предполагается при проверке выполнения задания.
